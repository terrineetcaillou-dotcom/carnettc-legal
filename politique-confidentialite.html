Politique de confidentialité

Application : Le Carnet de T&C
Dernière mise à jour : 5 juin 2026
Éditeur : Grégory Coulibaly

La présente politique explique comment l’application mobile Le Carnet de T&C (« l’Application », « nous ») traite vos données lorsque vous l’utilisez sur iPhone et, le cas échéant, sur Apple Watch.

1. Responsable du traitement

Grégory Coulibaly
Contact : heliozoaire@hotmail.com

2. Principe général

Le Carnet de T&C est un carnet d’entraînement. L’essentiel de vos données (séances, séries, programmes, notes) est enregistré localement sur votre appareil. Nous ne vendons pas vos données et n’utilisons pas de traceurs publicitaires.

3. Données traitées

3.1 Données d’entraînement (stockage local)

Sont stockées sur votre iPhone via la base de données de l’Application :

  • Séances, exercices, séries (poids, répétitions, durée, notes)
  • Programmes et modèles personnalisés
  • Historique, volumes, statistiques et badges calculés à partir de vos séances
  • Préférences d’affichage et réglages de l’Application

Ces données restent sur l’appareil sauf si vous utilisez volontairement une fonction d’export, de sauvegarde ou de synchronisation décrite ci-dessous.

3.2 Photos de progression (stockage local)

Si vous ajoutez des photos de progression, elles sont enregistrées sur votre appareil dans l’espace de l’Application. L’accès à la photothèque ou à l’appareil photo n’est demandé que lorsque vous choisissez d’importer ou de prendre une photo.

3.3 Apple Health (HealthKit) — optionnel

Avec votre autorisation explicite, l’Application peut :

  • Lire certaines données Health (ex. fréquence cardiaque, énergie active, distance) pour afficher des indicateurs
  • Écrire des séances d’entraînement dans Health lorsque vous activez cette option

Vous pouvez refuser ou révoquer l’accès à tout moment dans Réglages iOS → Santé → Accès aux données et appareils → Le Carnet de T&C. Les données Health restent gérées par Apple selon sa propre politique de confidentialité.

3.4 Notifications locales

L’Application peut programmer des notifications locales (fin de repos, rappels d’entraînement) sans serveur tiers pour ce faire. Vous pouvez les désactiver dans Réglages iOS → Notifications.

3.5 Widget et Apple Watch — optionnels

  • Le widget affiche des statistiques résumées via un conteneur partagé Apple (App Group) entre l’Application et le widget, toujours sur vos appareils.
  • L’Apple Watch, si utilisée, échange des données de séance avec l’iPhone via le système WatchConnectivity d’Apple (données liées à l’entraînement en cours).

3.6 Achats intégrés (Premium)

Les abonnements ou achats Premium sont gérés par Apple (StoreKit). Nous ne recevons pas vos coordonnées bancaires. Apple peut nous transmettre un identifiant d’achat anonymisé pour valider votre statut Premium.

3.7 Fonctions communauté et compte (optionnelles)

Certaines fonctions (partage de programmes entre amis, synchronisation cloud) ne sont actives que si vous vous connectez avec Sign in with Apple et utilisez nos services en ligne.

Dans ce cas, peuvent être traités :

  • Identifiant Apple (jeton d’identité fourni par Apple, non votre mot de passe Apple)
  • Pseudonyme ou prénom affiché si vous l’acceptez
  • Code d’invitation pour ajouter des amis
  • Programmes partagés et, le cas échéant, données de séance que vous choisissez de synchroniser

Ces données sont hébergées sur nos serveurs uniquement pour fournir le service demandé. Vous pouvez vous déconnecter depuis l’Application ; cela supprime les jetons locaux de session.

3.8 Données que nous ne collectons pas

  • Pas de publicité ciblée ni de revente de données
  • Pas de suivi publicitaire entre applications (pas de SDK analytics type Firebase Ads)
  • Pas de localisation GPS en arrière-plan pour suivre vos déplacements

4. Finalités

  • Permettre le suivi de vos entraînements
  • Afficher progression, historique et statistiques
  • Gérer le chrono de repos, les rappels et le widget
  • Intégrer HealthKit si vous l’autorisez
  • Gérer les achats Premium via Apple
  • Fournir les fonctions communauté / sync si vous les activez
  • Améliorer la stabilité de l’Application (journaux techniques locaux ou anonymes, sans profilage marketing)

5. Bases légales (RGPD)

  • Exécution du contrat / service : fonctionnement du carnet d’entraînement
  • Consentement : HealthKit, notifications, photos, Sign in with Apple, synchronisation cloud
  • Intérêt légitime : sécurité, prévention des abus sur les fonctions en ligne

6. Durée de conservation

  • Données locales : conservées tant que l’Application est installée ; supprimées si vous désinstallez l’app (sauf sauvegarde iCloud/iTunes selon vos réglages Apple)
  • Données serveur (compte) : conservées tant que votre compte est actif ; vous pouvez demander la suppression (voir section 8)
  • Exports : sous votre seule responsabilité une fois téléchargés

7. Destinataires et sous-traitants

  • Apple : distribution App Store, achats, HealthKit, notifications, iCloud (si activé par vous sur l’appareil)
  • Hébergeur serveur (si fonctions communauté/sync activées) : hébergement de l’API et de la base de données

Nous ne transmettons pas vos données à des courtiers de données publicitaires.

8. Vos droits

Conformément au RGPD, vous disposez des droits d’accès, de rectification, d’effacement, de limitation, d’opposition et de portabilité sur vos données personnelles, lorsque applicable.

Pour exercer vos droits : heliozoaire@hotmail.com. Vous pouvez aussi introduire une réclamation auprès de la CNIL.

9. Sécurité

Nous mettons en œuvre des mesures raisonnables (chiffrement en transit HTTPS pour les échanges serveur, stockage sécurisé des jetons sur l’appareil). Aucun système n’est toutefois infaillible ; évitez de partager vos exports ou codes d’invitation publiquement.

10. Mineurs

L’Application s’adresse aux personnes de 16 ans et plus. Elle n’est pas destinée aux enfants.

11. Modifications

Cette politique peut être mise à jour. La date en tête de page sera modifiée. En cas de changement important, nous pourrons vous en informer dans l’Application.
